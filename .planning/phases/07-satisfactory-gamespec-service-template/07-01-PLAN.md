```
[Unit]
Description=Satisfactory dedicated server
Wants=network-online.target
After=syslog.target network.target nss-lookup.target network-online.target

[Service]
Environment=SteamAppId=1690800
{auto_update_line}ExecStart={exec_start_path} -Port={port} -ReliablePort={reliable_port} -multihome=0.0.0.0 -log -unattended
User={user}
Group={user}
WorkingDirectory={working_directory}
Type=simple
Restart=on-failure
RestartSec=60
KillSignal=SIGINT
TimeoutStopSec=120

[Install]
WantedBy=multi-user.target
```

Key design notes:
- `{auto_update_line}` is either empty string (no auto-update) or a full `ExecStartPre=...` line with trailing newline. This avoids conditional template logic — caller provides empty or populated string.
- KillSignal=SIGINT (per SAT-02, research: SIGTERM kills without cleanup).
- TimeoutStopSec=120 allows graceful shutdown save.
- No `-players` flag — Satisfactory player limit is set in Game.ini, not CLI args.
- Port args placed BEFORE -log -unattended (research: args after those flags may be ignored).

```python
    """Set vm.max_map_count=262144 — prevents crash on large Satisfactory maps (SAT-07)."""
    console.print("Tuning vm.max_map_count for Satisfactory...")
    _write_via_sudo_tee(
```

2. **_render_satisfactory_service()** — Custom renderer for the Satisfactory template (it has different placeholders than the generic _render_service_file):
```python
def _render_satisfactory_service(
    *,
    user: str,
    working_directory: Path,
    exec_start_path: Path,
    port: int,
    reliable_port: int,
    auto_update: bool,
) -> str:
    """Render satisfactory.service.template with Satisfactory-specific placeholders."""
    template = _get_template("satisfactory.service.template")
    if auto_update:
        auto_update_line = (
            f"ExecStartPre=/usr/games/steamcmd +force_install_dir "
            f'"{working_directory}" +login anonymous +app_update 1690800 validate +quit\n'
        )
    else:
        auto_update_line = ""
    return template.format(
        user=user,
        port=port,
        reliable_port=reliable_port,
        exec_start_path=exec_start_path,
        working_directory=working_directory,
        auto_update_line=auto_update_line,
    )
```

```python
    _install_steamcmd()
```
