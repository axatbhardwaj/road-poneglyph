   legacy `sources.list`, the sed is a no-op — validate this NOW before the E2E actually
   runs so we know whether `_enable_debian_contrib_nonfree` needs a fallback.

   - Is `/etc/apt/sources.list` the primary source file (legacy)?
   - Or is there a `/etc/apt/sources.list.d/debian.sources` (deb822)?
   - Record in the E2E record — this determines whether Task 3 needs a code fix later.
